How to Play:

Try to guess the mystery word! This site will randomly select a word and display dashes in place of letters.

Guess each letter in the word to win. If you choose a letter that's not in the mystery word, you'll lose a guess and will need to select again.

If you make 10 incorrect guesses before you solve the mystery word, you'll lose the game and will almost certinly contract one of the game's infections.

Click the Start button to play. Select letters with the keybord.
